# Keywords

## Keywords

Values: **`True`**, **`False`**, **`None`**&#x20;

Conditions: **`if`**, **`elif`**, **`else`**&#x20;

Logical operators: **`and`**, **`or`**, **`not`**&#x20;

Loops: **`for`**, **`in`**, **`while`**, **`break`**, **`continue`**&#x20;

Functions:&#x20;

* **`def`**: for function declaration
* **`return`**: to return a value
